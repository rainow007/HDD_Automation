#Taffy_Web 

[Taffy](https://github.com/lovesoo/Taffy)自动化测试框架页面

# 1. 运行环境
- macOS，linux，windows
- python 2.7

# 2. 相关lib安装及配置
```
pip install flask
pip install flask-bootstrap
pip install flask-wtf
pip install nose
pip install nose-html-reporting
```

# 3.运行

```
$ python run.py
```

使用浏览器打开http://127.0.0.1:5000
